# Market Analysis

## Introduction

This document analyzes existing sports tracking applications in order to identify common features, strengths and limitations.

The objective is to understand the current market and identify opportunities for the proposed application.

---

## Existing Applications Comparison

| Application | Target Users | Main Features | Strengths | Limitations |
|---|---|---|---|---|
| Strava | Amateur, semi-professional and professional athletes | Activity tracking, performance statistics, training history, social features, route tracking | Large sports community, wide range of supported activities, external integrations | Requires an online account, depends on cloud infrastructure, many social features may not be needed by all users |
| Garmin Connect | Users with Garmin devices and athletes using wearables | Activity synchronization, health metrics, training plans, device integration | Strong integration with Garmin ecosystem and detailed health data | Best experience requires Garmin devices, dependency on external hardware |
| Nike Run Club | Mainly runners and fitness users | Run tracking, training plans, guided workouts | Simple user experience, good for beginners, running-focused features | Limited to running activities, less customization for other sports |

## Market Analysis Conclusions

Existing sports tracking applications provide many advanced features but usually depend on online services, user accounts and cloud infrastructure.

The proposed application focuses on a different approach:

- Local data storage.
- Minimal infrastructure requirements.
- Low operational cost.
- Simple MVP development.
- Future scalability.

The objective is not to replace existing platforms, but to provide a lightweight personal training management solution.
