# Market Analysis

## Introduction

This document analyzes existing sports tracking applications in order to identify common features, strengths and limitations.

---

# Strava

## Overview

Strava is a sports tracking platform focused on recording and analyzing physical activities.

## Main Features

- Activity tracking
- Performance statistics
- Training history
- Social features
- Route tracking

## Limitations

- Requires an online account
- Depends on cloud infrastructure
- Social features may be unnecessary for some users

---

# Garmin Connect

## Overview

Garmin Connect is a platform for managing data from Garmin devices.

## Main Features

- Activity synchronization
- Health metrics
- Training plans

## Limitations

- Strong dependency on Garmin ecosystem
- Requires compatible devices for full functionality

---

# Nike Run Club

## Overview

Nike Run Club focuses mainly on running activities.

## Main Features

- Run tracking
- Training plans
- Guided workouts

## Limitations

- Mainly focused on running
- Limited customization

---

# Conclusions

Existing applications provide many features but usually rely on online services.

The proposed application focuses on:

- Local data storage
- Low infrastructure cost
- Simple MVP approach
- Future scalability
